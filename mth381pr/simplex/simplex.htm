The Simplex Method

The Simplex Method is a logical process for solving Linear Programming problems consisting of three major steps:
 
 

  • Evaluate the criterion function at a vertex in the feasible solution (e.g., a point contained within the parameters of the constraints.)
  • Follow the edge in the direction of increasing (or decreasing for minimization problems) value until reaching the next vertex.
  • Repeat until no improvement occurs with movement from current vertex. This point is then an optimal solution.

 

*

Implementation of the Simplex Method involves:

  • finding a feasible solution
  • improving the solution until arriving at a point which yields an optimum solution

To ilustrate, take the equation:

P = 12B + 8W 

constrained by the three inequalities:

4B + 4W 1,400

6B + 3W 1,800 

2B + 6W 1,800
 

This example is taken from the book Five Golden Rules by John L. Casti. It is an optimization problem for making two dogfood blends (Bow-Wow brand(B)and Wuff-Wuff brand(W)) of lamb, beef and fish. The problem demands that we find values of B and W which are subject to the given inequalities (representing the amounts of lamb, beef and fish available) and that our solution is nonnegative. In order to create equalities from our inequalities for easier algebra, we introduce slack variables s1, s2 and s3. For example, the constraint equations are now transformed into the equalities:


4B + 4W + s1 =  1,400

6B + 3W + s2 =  1,800

2B + 6W + s3 = 1,800
 

In addition to having a nonnegative solution, we require that our variables B, W, s1, s2 and s3 be nonnegative, since a solution involving negative amounts of dogfood would be nonsense. The set of solutions to these equations is called a feasible solution. A basic solution is obtained by setting some of the variables equal to zero, thereby leaving a set of m variables in m equations. Also, the set of m non-zero variables is called a basis. We will be looking for a basic feasible solution for the dogfood problem. A company which Casti called the Chow-Down Company has to solve the problem in the given five variables (B, W, s1, s2 and s3 ) together with three constraint equations.

 

s1 =  1,400 - 4B - 4W

s2 =  1,800 - 6B - 3W

s3 = 1,800 - 2B - 6W

P = 12B + 8W 

Our first basic feasible solution is found by solving for our slack variables, making them the basis. Notice that each of the slack variables appeates in only one of the constraint equations. This is because for our slack variables to be a basis, they must be independent of one another. So, in order to use these variables to create a basic solution, we must set B and W equal to zero. This leaves us with a set of m=3 constraint equations in the three unknowns s1, s2 and s3. The solution to this set of equations can be read off as follows: s1=1,400,s2=1,800, and s3=1,800 which is suggested by the preceding equations.

Because the profit ,P, is zero at this point, we choose the variable with the largest coefficient in the criterion function to form the next basis; in this way, the profit will be increasing at the fastest possible rate. We will put B in the basis without letting it become larger than 300 (this would make s2 negative.) So, B=300, makes s2= 0 and B has become a basic variable and s2 has become a non-basic variable. Hence, the following set of equations, which expresses the basic variables s1, B, and s3 in the criterion function, P, in terms of the non-basic variables s2 and W:
 

 

s1 =  200 + (2/3)s2 - 2W

B = 300 - (1/6)s2 - (1/2)W

s3 = 1,200 - (1/3)s2 - 5W

P = 3,600 - 2s2 + 2W 
 

At this point, we know that the profit, P, is 3600, since both s2 and W are zero. Now, we increase the profit by interoducing W into a new set of basic variables, which cannot be greater than 100 (s1 would then be negative.) So, we set W=100 packages of Wuff-Wuff dogfood and replace s2 by W in our basis. This yields a new basis in B,W and s3. The new basis is:
 
 

W = 100 - (1/2)s1 + (1/3)s2

B = 250 + (1/4)s1 - (1/3)s2

s3 = 700 + (5/2)s1 - (4/3)s2

P = 3,800 - s1- (4/3)s2 

With s1 and s2 both less than zero in the profit equation, it is evident that an increase in either would only reduce the profit. Thus, we have arrived at our optimal solution. And, because the variables on the right hand side are all equal to zero, simply reading the constants will tell us the optimal solution: Chow-Down should produce 250 packages of Bow-Wow and 150 packages of Wuff-Wuff to give a gross profit of $3,800. It should be noted that the variable representing beef, s3, is inactive in the final equation, this implies that 700 pounds of beef are unused in the optimal mix of the brands. Conversely, s2 and s1 are both zero for the optimal solution, implying that Chow Down should use all of its stock in both lamb and fish compunds.
